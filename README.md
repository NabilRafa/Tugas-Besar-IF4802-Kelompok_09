# Tugas-Besar-IF4802-Kelompok_09
Rancanglah multi-linked list yang memodelkan Customer, Pakaian Laundry, dan hubungan diantara mereka. Seorang Customer dapat memiliki banyak pakaian laundry, dan satu pakaian laundry hanya dapat dimiliki oleh satu Customer. Program anda harus mengimplementasikan daftar dan semua fitur yang tercantuk di bawah ini.
